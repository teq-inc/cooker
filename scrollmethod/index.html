Scrollmethod scrollmethod.js

クリックで要素までスクロールさせるJQueryプラグイン。jquery.easing.jsが必要。

Usage

Markup

<body>
  <a href="#target">Drower toggle</a>
  <div id="target"></div>
</body>

JavaScript

$('a[href^=#]').scrollmethod({
    speed:     800,
    easing:   'easeOutExpo',
});
