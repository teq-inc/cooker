Usage

Markup

// Position Right
<div class="slidebar slidebar-right" data-width="240">
  <div class="slidebar-toggle">Slidebar Right</div>
  <div class="slidebar-body" style="width: 0;">
    <div class="slidebar-resize"></div>
      <div class="slidebar-body-inner">
        <h3>Slidebar Right</h3>
      </div>
    </div>
</div>

// Position Bottom
<div class="slidebar slidebar-bottom" data-height="300">
  <div class="slidebar-toggle">Slidebar Bottom</div>
  <div class="slidebar-body" style="height: 0;">
    <div class="slidebar-resize"></div>
    <div class="slidebar-body-inner">
      <h3>Slidebar Bottom</h3>
    </div>
  </div>
</div>

data-width,data-heightで開いた時のサイズを指定。

JavaScript

$('.slidebar').slidebar({
  position: 'left',
  speed: 500,
  easing: 'easeOutExpo',
  minSize: 10,
  parentClass: '.tab-content'
});

Methods

$().slidebar(options)

$().slidebar('open')

$('#element').slidebar('open')

$().slidebar('close')

$('#element').slidebar('close')

$().slidebar('toggle')

$('#element').slidebar('toggle')

$().slidebar('destroy')

$('#element').slidebar('destroy')
