Download on GitHub

row-table row-table.less

.row-tableはCSSのfloatプロパティは使用せずに、.row-tableクラスをdisplay: tableに設定。その子要素.col-*クラスにはdisplay: table-cellを設定することでコラムレイアウトを実装。
